---
title: Quick Config
description: Easy config and themes with @tamagui/config/v4
---

<IntroParagraph>
  With config v4, we've simplified the defaults and added a new helper function for generating themes, `createThemeSuite`.
</IntroParagraph>

Using it is as easy as importing the defaults:

```tsx fileName="tamagui.config.ts"
import { defaultConfig, defaultThemes } from '@tamagui/config/v4'
import { createTamagui } from 'tamagui'

export const config = createTamagui({
  ...defaultConfig,
  themes: defaultThemes,
})

type CustomConfig = typeof config

// ensure types work
declare module 'tamagui' {
  interface TamaguiCustomConfig extends CustomConfig {}
}
```

## Installation

DropdownMenu is already installed in `tamagui`, or you can install it independently:

```bash
yarn add @tamagui/dropdown-menu
```

if you want to use native Menus then make sure to install these depedencies in you expo directory

```bash
yarn add @react-native-menu/menu
```

```bash
yarn add react-native-ios-context-menu
```

```bash
yarn add react-native-ios-context-menu
```

## Anatomy

Import all parts and piece them together.

```jsx
import { Accordion } from 'tamagui';

export default () => (
    <DropdownMenu>
      <DropdownMenu.Trigger asChild>
        <Button/>
      </DropdownMenu.Trigger>

      <DropdownMenu.Portal zIndex={100}>
        <DropdownMenu.Content>
          <DropdownMenu.Item>
            <DropdownMenu.ItemTitle>About Notes</DropdownMenu.ItemTitle>
          </DropdownMenu.Item>
          <DropdownMenu.Item>
            <DropdownMenu.ItemTitle>Settings</DropdownMenu.ItemTitle>
          </DropdownMenu.Item>
          <DropdownMenu.Item>
            <DropdownMenu.ItemTitle>
              <Text>Calender</Text>
            </DropdownMenu.ItemTitle>
            <DropdownMenu.ItemIcon>
              <Calendar color="gray" size="$1" />
            </DropdownMenu.ItemIcon>
          </DropdownMenu.Item>
          <DropdownMenu.Separator />
          <DropdownMenu.Sub>
            <DropdownMenu.SubTrigger>
                <DropdownMenu.ItemTitle>Actions</DropdownMenu.ItemTitle>
            </DropdownMenu.SubTrigger>
            <DropdownMenu.Portal zIndex={200}>
              <DropdownMenu.SubContent
              >
                <DropdownMenu.Label fontSize={'$1'}>Note settings</DropdownMenu.Label>
                <DropdownMenu.Item onSelect={onSelect} key="create-note">
                  <DropdownMenu.ItemTitle>Create note</DropdownMenu.ItemTitle>
                </DropdownMenu.Item>
                <DropdownMenu.Item onSelect={onSelect} key="delete-all">
                  <DropdownMenu.ItemTitle>Delete all notes</DropdownMenu.ItemTitle>
                </DropdownMenu.Item>
                <DropdownMenu.Item onSelect={onSelect} key="sync-all">
                  <DropdownMenu.ItemTitle>Sync notes</DropdownMenu.ItemTitle>
                </DropdownMenu.Item>
              </DropdownMenu.SubContent>
            </DropdownMenu.Portal>
          </DropdownMenu.Sub>
        </DropdownMenu.Content>
      </DropdownMenu.Portal>
    </DropdownMenu>
)
```

### ItemIcon

Component to render Icon, for non native Menu you can pass an Icon component
for native Menus you can pass platform specific icons to the `android`/`ios` props

on iOS it renders platform [SF Symbol](https://hotpot.ai/free-icons?s=sfSymbols) icons

<PropsTable
  data={[
    {

      type: 'object',

    },
    {
      name: 'android',
      type: 'object',
      required: false,
    },
  ]}
/>

```jsx line=1
<DropdownMenu.ItemIcon
  ios={{
    name: '0.circle.fill', // required
    pointSize: 5,
    weight: 'semibold',
    scale: 'medium',
    // can also be a color string. Requires iOS 15+
    hierarchicalColor: {
      dark: 'blue',
      light: 'green',
    },
    // alternative to hierarchical color. Requires iOS 15+
    paletteColors: [
      {
        dark: 'blue',
        light: 'green',
      },
    ],
  }}
>
  <CircleIcon />
</DropdownMenu.ItemIcon>
```

### ItemIndicator

Use inside CheckboxItem or RadioItem to render when item is checked, so basically conditionally let you render a check mark

```jsx line=1
<DropdownMenu.ItemIndicator>
  <CheckmarkIcon /> {/* doesn't work with native prop */}
</DropdownMenu.ItemIndicator>
```

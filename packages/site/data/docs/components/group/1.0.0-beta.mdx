### Usage

```jsx
import { Group, Button } from 'tamagui'

export default () => (
  <Group>
    <Button>First</Button>
    <Button>Second</Button>
    <Button>Third</Button>
  </Group>
)
```

Group will pass it's border radius down. It ensures any items inside will get appropriately scaled border radius, subtracting 1px from inner items radius, to make sure it aligns.

### Sizing

The `size` property applies to radius, and passes to children:

```jsx
import { Group, Button } from 'tamagui'

export default () => (
  <Group size="$6">
    <Button>First</Button>
    <Button>Second</Button>
    <Button>Third</Button>
  </Group>
)
```

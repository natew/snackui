### Usage

The `value` property controls the percent, but you can override it by adjusting the `x` property.

```jsx
import { Progress, Button } from 'tamagui'

export default () => (
  <Progress value={60}>
    <Progress.Indicator animation="bouncy" />
  </Progress>
)
```

## Usage

```tsx
import { Tabs } from 'tamagui'

export default () => (
<Tabs defaultValue="tab1" width={400}>
  <Tabs.List space>
    <Tabs.Trigger value="tab1">Tab 1</Tabs.Trigger>
    <Tabs.Trigger value="tab2">Tab 2</Tabs.Trigger>
  </Tabs.List>

  <Tabs.Content value="tab1">
    <H5>Tab 1</H5>
  </Tabs.Content>
  <Tabs.Content value="tab2">
    <H5>Tab 2</H5>
  </Tabs.Content>
</Tabs>
)
```

### RovingIndicator

Used as an animatable indicator that can be shown upon selection, hover and focus. You may use several of these and combine `highlightMode`s to achieve the effect you want. Check [Animated Demo](#animated-demo) to see how it works action. Extends [ThemeableStack](/docs/components/stacks#themeablestack).

<Notice>
  You probably want to use `unstyled` on your [Trigger](#trigger) to disable hover/focus/active state on triggers if you use this component.
</Notice>
